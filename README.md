# AutoRename - Excel 批量重命名工具

这是一个基于 Python 的自动化办公工具，可以根据 Excel 名单自动修改文件夹内的文件名。

## 功能特点
- 🚀 **自动化**：一键匹配 Excel 名单与文件名。
- 🛠 **容错性**：自动处理文件后缀，支持多种格式。
- 📂 **易用性**：无需修改代码，直接输入路径即可使用。

## 如何使用
1. 安装依赖：`pip install pandas openpyxl`
2. 运行程序：`python main.py`
3. 按照提示输入 Excel 路径和目标文件夹路径。

## 技术栈
- Python 3.12.8
